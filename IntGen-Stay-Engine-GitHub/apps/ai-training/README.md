# Enterprise AI Training Platform

Static prototype for a unified enterprise AI training management system.

## What is included

- Admin overview with Kanban-style training assignment pipelines
- Department training cards for hours, roles, completion, and certification rate
- Learning path builder with avatar assignment, dependencies, and task bank
- Employee learning portal restyled into the same glassmorphism design system
- Assignment modal for allocating employees, departments, hours, and training scope
